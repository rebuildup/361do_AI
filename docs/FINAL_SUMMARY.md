## 🚀 デプロイメント

### 開発環境

```bash
# フロントエンド開発サーバー
cd frontend
npm run dev

# バックエンド (既存)
python main.py --ui fastapi
```

### プロダクション環境

```bash
# 統合デプロイメント
docker-compose up --build

# または個別ビルド
docker build -f Dockerfile.integrated -t ai-agent-integrated .
docker run -p 80:80 ai-agent-integrated
```
